# ATM Machine Project (C++)

## 📌 Overview

This is a console-based ATM Machine simulation built using C++. The project allows users to securely access their account using a PIN and perform basic banking operations such as checking balance, depositing money, and withdrawing funds.

This project was developed to practice core C++ programming concepts including functions, loops, conditional statements, user input handling, and menu-driven applications.

---

## 🚀 Features

- 🔐 PIN Authentication System
- 🔒 Account lock after 3 failed login attempts
- 💰 Check account balance
- 💵 Deposit money
- 🏧 Withdraw money
- ✅ Input validation
- 📋 Menu-driven interface
- 🔄 Continuous operation until user exits

## 💻 How to Run

### Compile the Program

```bash
g++ atm_machine.cpp -o atm
```

### Run the Program

**Windows**

```bash
atm.exe
```

**Linux / macOS**

```bash
./atm
```
